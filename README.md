## A React / NodeJS - Redux voting application, built using the following technologies:

#### React
#### NodeJS
#### Redux
#### ImmutableJS
#### Mocha
#### Chai

## Instructions for use:

git clone `https://github.com/jackhardy1/react-redux`

To install necessary dependencies, run:

`npm install`

## Running the application

To run the application, first start the NodeJS server-side application:

`cd voting-server && npm run start`

Then run the frontend ReactJS application:

`cd ../voting-client && npm run start`

visit `localhost:8080`


## To run tests:

#### Server
From react-redux root folder,
`cd voting-server && npm run test`

#### Client
From react-redux root folder,
`cd voting-client && npm run test`
